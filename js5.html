Count Characters
    You are given a string S, and your task is to return an array B (having a size of 2), 
    where B[0] contains the count of character A (uppercase) in string S and B[1] 
    contains the count of character D (uppercase) in string S.

    Note: You have to complete countCharacters function. No need to take any input.
    
        const countCharacters = (S) => 
            {
                let charD=0, charA = 0; B = [];
                for (let i = 0; i < S.length; i++) {
                if (S[i] === "A") {
                charA++;
            } else if (S[i] === "D") {
            charD++;
            }
            }
            B[0] = charA;
            B[1] = charD;
        return B;
        };

Count the Vowels and Consonants
        You are given a string S containing both uppercase and lowercase letters. 
        You need to find out the number of vowels and the number of consonants in the given string.
        
        Note: You have to complete Count_Vowels function and Count_Consonants function. No need to take any input.
    
        var Count_Vowels= (S) => 
        {
            vowel=0;
            S=S.toLowerCase();
            for(let i=0; i<=S.length; i++){
                if(S[i]=="a" || S[i]=="e" || S[i]=="i" || S[i]=="o" || S[i]=="u"){
                vowel++;
                }
            }
            return vowel;
        };
        var Count_Consonants= (S) => 
        {
            let consonants=0;
            S=S.toLowerCase();
            for(let i=0; i<=S.length; i++){
                if(S[i]!="a" || S[i]!="e" || S[i]!="i" || S[i]!="o" || S[i]!="u"){
                    consonants++;
                }
            }  
            return consonants;
        };
